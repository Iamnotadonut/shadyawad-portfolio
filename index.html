Technical Details

This portfolio website is built with vanilla JavaScript, CSS, and HTML for optimal performance. It detects the user's system color scheme via prefers-color-scheme on first load, with a manual toggle that persists via localStorage.

The responsive, mobile-first design features a green theme, subtle animations (fade-ins and scale effects) via Intersection Observer, an extensible project section (duplicate the project-card div), a connect section for social profiles (LinkedIn, Twitter/X, GitHub), and a pricing calculator with a smooth slider and styled form and result boxes for estimating web development costs. No external frameworks ensure a lightweight, fast-loading site.
